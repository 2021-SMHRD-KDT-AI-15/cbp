금연가이드

금단증상 극복하기

4D 방법

Delay : 잠깐 시간가지기
Drink : 물 마시기
Distraction : 다른일에 집중하기
Deep breathing : 심호흡하기

2R 방법

Renimd : 상상하기
Refuse : 거절하기

짜증, 신경질

1. 운동하기
2. 술과 카페인이 들어간 음료 줄이기
3. 충분한 수면 취하기
4. 심호흡하기

불면증

1. 자기 전 산책
2. 독서
3. 따뜻한 목욕
4. 바나나 또는 따뜻한 우유 섭취
5. 점심 이후에는 카페인 음료 피하기

식욕증가

1. 비상상자 만들기
- 입이 심심할 때 십을 수있는 것들을 넣어두기
- 빨대, 계피줄기, 커피젓는 막대, 이쑤시개, 껌, 신선한채소 등
2. 물이나 저칼로리 음료 마시기
3. 금연 초기 술자리는 가급적 피하기

집중력감소/피로감

1. 물 많이 마시기
2. 빠른 걸음으로 야외 산책하기
3. 며칠간은 스케줄을 최소화하기
4. 휴식시간 가지기
5. 매일 충분히 자고, 낮잠자기

변비, 가스, 복통 등

1. 섬유소가 많은 음식과 수분 섭취
- 과일, 생야채, 곡물, 시리얼
2. 식습관 변화시키기
3. 영양사나 의사와 상담
